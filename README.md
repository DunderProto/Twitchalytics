# Twitchalytics

Twitchalytics is an online application that showcases the statistics of the popular streaming website known as Twitch. You can view the live version [here.](https://dunderproto.github.io/Twitchalytics/)

## Technologies
- HTML 5
- CSS 3
- JavaScript
- D3.js
- Webpack (4.43.0)
